「笔记」Minecraft 我的世界 Fabric 模组入门

发表于2024-02-24|更新于2024-02-24|笔记
|字数总计:214|阅读时长:1分钟|阅读量:

配置 Fabric 开发环境

  1. 自行下载安装 IDEA，破解可参考另一篇文章
  2. 安装 Minecraft Development 插件，直接在 IDEA 插件商店搜索安装即可

新建一个 Fabric 项目

创建一个新项目，选择 Minecraft Development 模板，根据向导填写相关内容

编写第一个 Fabric 模组

参考Fabric Wiki

注意事项

  1. 修改Gradle设置中运行、测试、构建为 IDEA
  2. 根据需求修改fabric.mod.json文件中 depends
  3. 记得在resources目录下，添加assets/modid/icon.png
  4. 根据提示，启动脚本Minecraft Client 添加 -Dfabric-api.networking.force-packet-serialization=false

遇到的问题

Out of Memory

  1. 修改 IDEA 使用内存大小
  2. 修改 gradle.properties 文件中 org.gradle.jvmargs=-Xmx4G

[Download-2/ERROR] (Minecraft) Failed to fetch user properties

具体哪步弄好的不知道，应该是缺少一些东西，重新 build build dependencies getSources 后好了
