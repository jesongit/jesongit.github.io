「笔记」Markdown ---- 一款轻量级标记语言

发表于2020-04-01|更新于2023-03-11|笔记
|字数总计:976|阅读时长:4分钟|阅读量:

介绍

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。

常用语法

效果 Markdown 快捷键
粗体 **text** Ctrl/⌘ + B
斜体 *text* Ctrl/⌘ + I
超链接 [title](http://) Ctrl/⌘ + K
内嵌代码 `code` ⌃ + `
图片 ![alt](http://) Ctrl/⌘ + Shift + I
列表 * item Ctrl + L
Blockquote > quote Ctrl + Q
一级标题 # title ⌘ + 1
二级标题 ## title ⌘ + 2
三级标题 ### title ⌘ + 3

更多语法

图片展示及快捷键来自 Typora ，在不同地方可能存在差异
